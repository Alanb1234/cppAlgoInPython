Python vs C++

This is a demonstration to show how using libraries that allow for interoperation between c++ and pyton can make your python programs more fast and efficient.

To demonstrate this, execution times of some sorting algorithms written using python alone and C++ operation with python have been are being compared.

Select the sorting algorithm and the number of elements to be sorted, then submit to generat the execution time to sort the random arrays and illistrations of the arrays. You will see that the run time for the pyton alone is usual an order of magnitude greater then that for the C++.
